What is visual studio code and how to set it up:

Visual studio code is an offline code editor, meaning it can work without internet. It is free of cost and it is developed by Microsoft. First, to set up visual studio code, we have to download the extensions of what programming languages you are going to code in. Then, after the exntensions are downloaded, we can make a folder in file explorer and open it in visual studio code. You can create optional files like "index.html", "main.js", and "style.css".



The console screen helps us to debug programs by testing lines of code ony by one. This can help in finding issues in code. The console can also help us to give notes to the person who is reading our code so they understand what segment of code has what functionality. We can also print variables and check the type of a particular piece of data. To print in the console, use this code: console.log();. In the brackets, just add the content you want to add.
